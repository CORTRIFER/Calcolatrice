Istruzioni Calcolatrice Scientifica

Operazioni di base

  • Utilizza i pulsanti numerici per inserire i numeri
  • Usa i pulsanti +, -, *, / per le operazioni aritmetiche
  • Premi = per calcolare il risultato
  • Usa C per cancellare tutto
  • Premi . per inserire un numero decimale

Funzioni scientifiche

  • Funzioni trigonometriche: Seleziona dal menu a tendina "Trg" per sin, cos, tan
  • Logaritmi:
    • Premi log per inserire un logaritmo
    • Puoi specificare la base (inserisci 'e' per il logaritmo naturale)
  • Radici:
    • Premi √ per inserire una radice
    • Specifica l'indice della radice (2 per radice quadrata, 3 per cubica, ecc.)
  • Potenza:
    • Premi x^y per inserire un'esponenziazione
    • Specifica base ed esponente

Conversione in frazione

  • Premi Converti numToFrac per convertire un numero decimale in frazione
  • Premi di nuovo per tornare al formato decimale

Tastiera(Per Desktop only)

  • Puoi usare la tastiera per inserire numeri e operatori
  • Enter o = per calcolare
  • Backspace per cancellare un carattere
  • c oppure C o Esc per cancellare tutto
  • Scrivi sin, cos, tan per le funzioni trigonometriche
  • Scrivi log, root, pow per le altre funzioni
  • Scrivi ntf per convertire in frazione
  • In questo caso si possono usare le parentesi(o)per i calcoli

Note importanti

  • Le funzioni trigonometriche usano i radianti
  • In caso di errore, il display mostrerà "Errore"
  • Le parentesi non chiuse vengono chiuse automaticamente al calcolo
